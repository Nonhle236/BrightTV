# BrightTV

Project Overview

This project analyzes Bright television viewership data. The goal is to uncover trends in viewership behavior, including when people watch TV, what they watch, and how this varies across differet channels.

What was done:
Combine multiple datasets into a single structured dataset
Clean and transform raw data for analysis
Analyze viewership patterns by:
Time (hour, day, weekday vs weekend)
Demographics (age, gender, race, province)
Channels and content popularity


Finally, a presentation is prepared to show the data analysis.
